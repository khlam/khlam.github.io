Projects

Depth Sensing with Computer Vision and Lidar (2018)

My university capstone project was to build a multi-dimensional depth sensor using computer vision and a single-dimension lidar sensor. My final system integrated a wooden jig, pre-trained YOLO model, 4K Logitech webcam, and an RPlidar A1 LiDAR sensor. github.com/DSCVL Paper (Github, LaTeX)

Depth Sensor System Demo (youtu.be/6v8h8LPFRls)

Oregon State University Explainable AI Research (2018-2021)

Our research explored explanation-informed acceptance testing of deep adaptive programs. I designed explanation-rich scenarios for research, managed a graduate-student team, coordinated, designed, and ran human-subject studies. The following is video of the game I designed for our research, a simplified version of Starcraft 2. Inspired by the popular mod "Nexus Wars", this experimental domain features an adversarial multi-agent dynamic, discrete action space, continuous state space, and a partially observable environment.
